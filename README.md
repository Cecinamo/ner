<h1>Loko - NER</h1><br></html>


 **NER** is a Loko extension dealing with Named Entity Recognition models. 
 
It provides **Spacy pretrained models** and a <strong>base rule</strong> model which can be used to predict entities without training. Otherwise, in few steps, you can train new models starting from the **HF** or **Spacy** models in order to predict your custom entities. 

You can `create`, `delete`, `import`, `export` or make a `copy` of you models directly from the **NER GUI**:
